# MeasureOS_2 - Simple Version

##  Description
This is a simple implementation of Exercice 3 (Series 2).  
It simulates monitoring a patient’s temperature and triggering treatment based on thresholds.

---

##  Objective
- Read temperature values
- Detect threshold conditions (S1, S2)
- Execute treatment function accordingly
